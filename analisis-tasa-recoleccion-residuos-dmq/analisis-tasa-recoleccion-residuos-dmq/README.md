# Análisis de la tasa de recolección de residuos sólidos en el DMQ

Proyecto integrador final individual.

## Objetivo
Evaluar la eficiencia, equidad e incentivo ambiental de tres mecanismos:
1. Cobro asociado al agua.
2. Cobro asociado a la electricidad.
3. Cobro independiente según residuos estimados.

## Importante
Los datos iniciales son simulados y están identificados como tales.

No necesitas Python. Abre `dashboard/index.html` con Live Server en VS Code.
