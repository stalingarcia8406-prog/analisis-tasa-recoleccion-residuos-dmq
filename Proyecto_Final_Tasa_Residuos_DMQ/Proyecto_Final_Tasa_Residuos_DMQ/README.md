# Evaluación y rediseño de la tasa de recolección de residuos sólidos del DMQ

Proyecto integrador final individual desarrollado mediante arquitectura multiagente.

## Modalidad
Análisis de implementación y rediseño de política pública tarifaria.

## Componentes
- Informe académico en Word y PDF.
- Dashboard web compatible con Vercel.
- Datos simulados y diccionario de variables.
- Arquitectura multiagente, prompts y registro de decisiones.
- Presentación para la defensa.
- Guion del video de cinco minutos.
- Documentación para GitHub y Vercel.

## Advertencia metodológica
Los microdatos incluidos son simulados para demostrar la metodología. No deben presentarse como cifras oficiales. Las fuentes oficiales deben incorporarse y verificarse antes de la entrega definitiva.

Abra `03_dashboard/index.html` con Live Server en Visual Studio Code.
