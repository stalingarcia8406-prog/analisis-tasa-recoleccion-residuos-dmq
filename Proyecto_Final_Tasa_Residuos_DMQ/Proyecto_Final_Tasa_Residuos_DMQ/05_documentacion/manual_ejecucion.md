# Manual de ejecución

## Dashboard
1. Abra la carpeta en Visual Studio Code.
2. Instale la extensión Live Server.
3. Abra `03_dashboard/index.html`.
4. Pulse `Go Live`.

## GitHub
Suba el contenido del proyecto, no el ZIP.

## Vercel
1. Importe el repositorio.
2. Configure `03_dashboard` como Root Directory.
3. Framework Preset: Other.
4. Deploy.

## Datos
La base inicial es simulada. Reemplácela con fuentes oficiales cuando estén disponibles.
