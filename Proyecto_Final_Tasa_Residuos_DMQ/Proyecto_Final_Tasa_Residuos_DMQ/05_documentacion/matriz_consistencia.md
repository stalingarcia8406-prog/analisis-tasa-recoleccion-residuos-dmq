# Matriz de consistencia

| Elemento | Contenido |
|---|---|
| Problema | Desvinculación entre pago y generación de residuos |
| Pregunta | ¿Cómo afecta el mecanismo de cobro a equidad y eficiencia? |
| Objetivo | Comparar tres escenarios y proponer rediseño |
| Unidad de análisis | Usuario simulado |
| Indicadores | Recaudación, pago medio, cobertura y alineación pago-residuos |
| Método | Revisión documental, análisis descriptivo y simulación |
| Alcance | Evaluación de implementación; no causal |
